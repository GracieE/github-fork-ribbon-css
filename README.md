# "Made at General Assembly" CSS ribbon

This is a recreation of the [Fork me on GitHub ribbon](https://github.com/blog/273-github-ribbons)
in CSS, hence resolution-independent.

See 'em in action! [http://GracieE.github.io/github-fork-ribbon-css/](http://GracieE.github.io/github-fork-ribbon-css/)
